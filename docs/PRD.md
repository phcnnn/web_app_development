# 快速小資記帳 (QuickSpend) PRD

## 1. 專案概述

- **背景與動機**
  許多使用者需要一個極簡的工具來紀錄每日開銷，不希望被複雜的報表或過多的欄位干擾。本專案旨在提供一個快速、有效的記帳介面，讓使用者專注於最重要的收支紀錄。
- **目標用戶**
  需要輕量記帳工具、希望快速完成紀錄的使用者。
- **核心價值主張**
  簡單三步驟（輸入、分類、存檔），讓記帳成為日常生活中零負擔的小習慣。

## 2. 功能需求

- **1. 新增消費紀錄**
  - 作為使用者，我希望可以輸入「金額」、選擇「類別」（如餐飲、交通、購物等）並撰寫「備忘內容」，以便詳實紀錄每一筆消費詳情。
- **2. 刪除消費紀錄**
  - 作為使用者，我希望可以針對輸入錯誤或不再需要的紀錄進行「刪除」，以維持帳務清單的正確性。
- **3. 查看當月總支出**
  - 作為使用者，我希望可以在系統首頁或顯眼處即時看到「當月累積總支出金額」，以便掌握本月的預算使用進度。
- **4. 帳務紀錄列表**
  - 作為使用者，我希望可以條列式檢視所有的歷史消費紀錄，以便核對支出。

## 3. 非功能需求

- **技術限制**
  - 後端：Python Flask。
  - 前端：Jinja2 模板 + CSS 樣式。
  - 資料存儲：SQLite。
- **效能考量**
  - 確保月度支出統計的計算與顯示能在 1 秒內完成。

## 4. MVP 範圍

- **Must Have (必須有)**
  - 消費紀錄的新增（金額、類別、備忘）。
  - 消費紀錄的刪除按鈕與功能。
  - 首頁即時顯示當月總支出總計。
  - 基礎紀錄列表。
- **Should Have (應該有)**
  - 編輯現有紀錄的功能。
  - 按類別顯示統計佔比。
